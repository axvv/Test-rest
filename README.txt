GET запрос по id продукта
	Заполнены все поля валидными данными
	успешность запроса
 	в теле ответа id равен тому что был в запросе
 	в теле ответа price больше 1
 	в теле ответа CategoryTitle заполнен
GET запрос по категории продукта
	Заполнены все поля валидными данными
 	в теле ответа id равен тому что был в запросе
 	в теле ответа title равен тому что был в запросе
 	в теле ответа CategoryTitle заполнен в каждом продукте этой категории
